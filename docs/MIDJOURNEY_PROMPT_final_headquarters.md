## Using references on the Midjourney website (no typed flags needed)

1. Click the **image icon (+)** at the left of the prompt bar and upload two images: `hq-concept.png` and Rollo's grass photo.
2. Hover each uploaded thumbnail — small icons appear letting you set its role. Set the **concept render as a Style reference** (palette icon) and **Rollo's photo as a Character/Omni reference** (person icon).
3. Open the **settings sliders** next to the prompt bar: set aspect ratio wider than square (3:2 or Landscape) and choose **Raw** mode if offered.
4. Type the short prompt, run, reroll freely.

## How to run it

1. **Anchor the soul:** add `--sref` with the URL of the current concept (`hq-concept.png` uploaded to Midjourney) so the new render keeps the mood we approved.
2. **Rollo fidelity:** add a character/omni reference (`--cref` or `--oref` depending on your MJ version) pointing at the grass-sunbathing photo from `art/photos/rollo/`. If the dog still isn't *him*, render the scene with the generic collie and we'll fix Rollo alone with region-vary + the reference — his face matters more than any other pixel.
3. **Expect text to sneak in.** Midjourney adores writing gibberish on signs. "Blank parchment" phrasing fights it; reroll or region-erase any lettering that appears. Text-free is non-negotiable — every word in the app is a live overlay.
4. Upscale the winner to maximum, drop it into `art/concepts/` as `hq-final-v1.png`, and I'll remeasure every hotspot and overlay against the new pixels (properly this time, with the measuring tool, not my eyeballs).
